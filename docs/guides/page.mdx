# Guides

## Prompt Flows

Use `bean.group()` or `bean.fluent()` when you need multi-step interactive workflows.

## Spinners and Progress

Use `bean.spinner()` for single activity feedback and `bean.createMultiProgress()` for concurrent tasks.

## Output Formatting

Use `bean.table()`, `bean.tree()`, `bean.json()`, and `bean.diff()` for structured output.
